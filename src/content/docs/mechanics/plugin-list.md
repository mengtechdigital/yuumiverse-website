Yuumiverse runs on a mix of **9 custom in-house plugins** and **35+ trusted public plugins** to deliver a unique semi-vanilla experience.

## Custom Plugins (In-House)

These are built specifically for Yuumiverse and documented right here in this wiki.

| Plugin | What It Does |
|--------|--------------|
| [Anglers](/plugins/anglers/) | Fishing overhaul with built-in anti-auto-fishing |
| [CropFarm](/plugins/cropfarm/) | Custom crops and farming expansions |
| [Marriage](/plugins/marriage/) | SMP marriage system with roles and perks |
| [Nightfall](/plugins/nightfall/) | Brutal nights with runner zombies and buffed mobs |
| [Realms](/plugins/realms/) | Town creation, land claims, raids, and diplomacy |
| [StackPlus](/plugins/stackplus/) | 99-item stacks to reduce inventory clutter |
| [Terraform](/plugins/terraform/) | Excavators, wands, tunnelers, and terraforming tools |
| [WildHarvest](/plugins/wildharvest/) | TreeFeller and VeinMiner for natural blocks |
| [Yuumi](/plugins/yuumi/) | Attach to players as a floating companion |

## Public Plugins

We also use well-known public plugins for performance, security, chat, world management, and quality of life.

### Protection & Logging
- **CoreProtect** — Block logging and rollbacks
- **CoreProtect-additions** — Extra features for CoreProtect
- **CoreProtectTNT** — TNT-specific logging
- **DimensionControl** — Dimension management
- **LuckPerms** — Advanced permission system
- **LuckPermsGUIPlus** — GUI editor for LuckPerms
- **nLogin** — Authentication and login security

### Chat & Social
- **DiscordSRV** — Two-way Minecraft ↔ Discord chat bridge
- **Essentials** — Core commands, homes, warps, and kits
- **GSit** — Sit, lay, and pose anywhere
- **LPC** — Chat formatting and prefixes
- **PlaceholderAPI** — Placeholder system for other plugins
- **SkinsRestorer** — Custom player skins
- **TAB** — Tab list and nametag customization
- **voicechat** — Proximity voice chat in-game

### World & Building
- **Citizens** — NPC creation and management
- **ChunkyFast** — Fast world pre-generation
- **dynmap** — Live web-based world map
- **FastAsyncWorldEdit** — Advanced world editing
- **Multiverse-Core** — Multiple world management
- **Multiverse-Portals** — Portal creation between worlds

### Performance & Compatibility
- **PlugManX** — Plugin manager (Paper)
- **ViaVersion** — Allow newer clients to connect
- **ViaBackwards** — Allow older clients to connect

### Gameplay & Fun
- **DropHead** — Player and mob head drops
- **Drugs** — Custom consumable items (server-specific)
- **ScNickeo** — Server-specific feature plugin
