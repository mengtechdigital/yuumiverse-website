## Is Yuumiverse pay2win?

**No.** There are no ranks, crates, or purchasable advantages. Our donation store (if we have one) only sells cosmetics. Everyone plays on the same footing.

## Do I need mods to play?

**No.** Yuumiverse is entirely plugin-based. Just connect with vanilla Minecraft 1.21 Java Edition.

## Can I play on Bedrock?

**Not currently.** We only support Java Edition 1.21.

## What plugins does the server have?

We run 9 custom plugins built specifically for Yuumiverse:

- [Yuumi](/plugins/yuumi/) — Companion attachment
- [Anglers](/plugins/anglers/) — Fishing overhaul
- [CropFarm](/plugins/cropfarm/) — Custom crops
- [Marriage](/plugins/marriage/) — SMP marriage system
- [Nightfall](/plugins/nightfall/) — Hard nights
- [Realms](/plugins/realms/) — Towns/claims/raids
- [StackPlus](/plugins/stackplus/) — 99 item stacks
- [Terraform](/plugins/terraform/) — Terraforming tools
- [WildHarvest](/plugins/wildharvest/) — TreeFeller + VeinMiner

## How do I claim land?

Use the [Realms](/plugins/realms/) plugin. Type `/realm create <name>` to start your own realm, then `/realm claim` to claim chunks around you.

## Can I raid other players?

Yes, but only during **declared wars** between Realms. Random griefing is against the rules. See [Rules](/getting-started/rules/) and [Realms](/plugins/realms/) for details.

## How do I report a bug or player?

Open a ticket in our [Discord](https://discord.gg/yuumiverse) or message a mod in-game.

## Is there a world border?

Yes. The world border is set to give plenty of room while keeping everyone reasonably close for interaction.

## Do you reset the world?

We prefer not to. Builds and history matter here. Major changes are announced well in advance.
