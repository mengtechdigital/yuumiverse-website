## Server Address

| Setting | Value |
|---------|-------|
| **IP** | `play.yuumiverse.gg` |
| **Version** | Java Edition 1.21 |
| **Modded?** | No — pure plugins |

## Step-by-Step

1. Open the Minecraft Launcher
2. Select **Java Edition** and launch version **1.21**
3. Click **Multiplayer** → **Add Server**
4. Enter `play.yuumiverse.gg` as the Server Address
5. Click **Join Server**

## First Time Here?

- Type `/help` to see available commands
- Check `/realm` to learn about claiming land
- Read up on our [plugins](/plugins/yuumi/) — they change a lot of vanilla mechanics
- Say hi in chat! We're a friendly community
